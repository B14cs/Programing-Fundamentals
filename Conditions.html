What are Conditions?
Conditions allow your program to make decisions and execute different code based on whether something is true or false. Just like in real life: "IF it's raining, THEN take an umbrella, ELSE leave it at home."

Comparison Operators

Tools for Making Comparisons

üí° Tip: Comparison operators return boolean values (true or false) based on the comparison

Equal to

==
5 == 5 // true
"hi" == "hi" // true
3 == 5 // false

Strict Equal

===
5 === 5 // true
5 === "5" // false
(checks type too)

Not Equal

!=
5 != 3 // true
"hi" != "hi" // false
10 != 10 // false

Greater Than

>
10 > 5 // true
3 > 7 // false
5 > 5 // false

Less Than

<
3 < 7 // true
10 < 5 // false
5 < 5 // false

Greater or Equal

>=
10 >= 5 // true
5 >= 5 // true
3 >= 7 // false

Less or Equal

<=
3 <= 7 // true
5 <= 5 // true
10 <= 5 // false

Logical Operators

Logical Operators - Combining Conditions

AND (&&)

Both conditions must be true

A B A && B
true true true
true false false
false true false
false false false
if (age >= 18 && hasLicense) {
  // Can drive
}

OR (||)

At least one condition must be true

A B A || B
true true true
true false true
false true true
false false false
if (isWeekend || isHoliday) {
  // No school!
}

NOT (!)

Reverses the condition

A !A
true false
false true
if (!isRaining) {
  // Go outside
}
